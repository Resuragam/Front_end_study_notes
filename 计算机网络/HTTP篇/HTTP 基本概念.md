	## HTTP 是什么？
---
HTTP 是超文本传输协议，也就是**HyperText Transfer Protocol**。

```ad-question
能否详细解释「超文本传输协议」？
```

HTTP 的名字 「超文本传输协议」，可以拆分为三个部分：

* 超文本
* 传输
* 协议
